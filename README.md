## Prerequisites

### Installed Software 

- HomeBrew
- Parcel web package manager (Link: https://parceljs.org/getting_started.html)
- Web browser

### Editors 

- Visual Studio Code or Atom or any other text editor of your choice 

## Pull dependencies

```
$ npm install
```

## Run Code

```
$ npm run dev
```
